# ManuQueue Capture Extension

Minimal Chrome extension for Vendor Capture v2.

## Load Unpacked

1. Open `chrome://extensions`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select this `extension/` folder.

## Use

1. Login to ManuQueue as `ADMIN`.
2. Open extension popup and set your ManuQueue base URL.
3. On a vendor page, click `Capture Current Tab`.
4. In ManuQueue `/orders/new`, click `Use Latest Capture`.
